# test-task
Реализация ProcFS модуля ядра, который включает функционал создания сетевого интерфейса с генерацией случайного MAC-адреса и назначением переданного IPv4 адреса. Сетевой интерфейс создается в момент передачи IPv4 адреса в ProcFS ноду. Отключить сетевой интерфейс можно с помощью передачи нуля ("0") в ноду.

Проверка функционала:
1. `make`
2. `make load` - загрузит собранный модуль через insmod
3. Записать в `/proc/vnet_ping_ip` IPv4 адрес для сетевого интерфейса
4. Выполнить команду `ping` для использованного IPv4 адреса
